# Flight Prediction Algorithm Comparison

This project compares ML models (e.g., Random Forest vs. XGBoost) for predicting flight prices/delays using a Kaggle dataset.

## Setup
- Install: `pip install pandas scikit-learn xgboost`
- Run: `python flight_prediction.py`

## Models Compared
- Logistic Regression
- Random Forest
- XGBoost (best performer!)
- Neural Network

Check `flight_prediction.py` for the code.
